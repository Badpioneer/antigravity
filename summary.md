# Project Summary: Brand Landing Page

## Overview
This project is a high-end, modern landing page built using the latest web technologies. It features a visually stunning design with focus on performance and interactive user experience.

## Tech Stack
- **Framework**: React 19
- **Build Tool**: Vite 8
- **Styling**: Tailwind CSS v4 (latest version)
- **Language**: TypeScript

## Key Features
1. **Premium Hero Section**: A high-impact centered layout with modern typography and gradient headers.
2. **Animated Mesh Gradient**: A full-screen background with smooth, animated radial gradients for a dynamic "live" effect.
3. **Glassmorphism Design**: Use of translucent backgrounds and backdrop blurs in the navigation and buttons.
4. **Interactive Components**:
   - **Custom Button**: A reusable component with multiple variants (Primary, Secondary, Glass).
   - **Click Counter**: A real-time state-managed counter.
   - **Reset Functionality**: A dedicated button to clear the counter state.
5. **Modern Layout**:
   - Sticky, translucent Navbar.
   - Fixed, full-screen background that adapts to any resolution.
   - Perfectly centered main module.

## Setup and Running
To run this project locally:

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```

3. **View the App**:
   Navigate to [http://localhost:5173/](http://localhost:5173/)

## Project Structure
- `src/components/`: Reusable UI components (Button, Navbar).
- `src/App.tsx`: Main landing page logic and layout.
- `src/index.css`: Tailwind v4 configuration and custom animations.
